# Data-Structures
Implementation of various data structures

# PA-1 has the implementation of:
  - Doubly Linked lists
  - Stacks
  - Queues
  - Stack Sorting
  
# PA-2 has the implementation of:
  - General trees
  - AVL and BST
  - Expression trees
  
# PA-3 has the implementation of:
  - Heaps
  - Hash Tables
  - Merge sort and Insertion sort
  - Tries
  
# PA-4 has the implementation of:
  - Graphs (Incomplete)
